# Backend相关介绍

## 部署方式
1. 下载项目源代码后，执行
  ```bash
  pip install -r requirements.txt
  ```
  安装项目相关依赖。

2. 运行项目
  ```bash
  python manage.py runserver
  ```
3. 默认端口为8000，访问http://localhost:8000/ 即可。
